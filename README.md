JSON Contact Management System
====
📌 Project Overview
====
The JSON Contact Management System is a Python-based Command Line Interface (CLI) application that allows users to manage contacts efficiently. The project stores contact information in a JSON file and demonstrates Python concepts such as File Handling, JSON Operations, Exception Handling, Custom Exceptions, and Logging.

🚀 Features
====
Add new contacts
View all contacts
Search contacts by name
Delete contacts
Store data in a JSON file
Handle invalid user inputs using Exception Handling
Use Custom Exceptions for contact-related errors
Log important events and errors to a log file
Gracefully handle missing or corrupted JSON files
